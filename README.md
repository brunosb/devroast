# 🔥 DevRoast

Seu código é ruim? A gente prova.

**DevRoast** é um app web onde você cola seu código e recebe um _roast_ — uma avaliação brutal e bem-humorada gerada por IA. Cada trecho recebe uma nota de 0 a 10, e os piores vão parar no **Leaderboard da Vergonha**.

> Projeto construído durante o evento **NLW** da [Rocketseat](https://rocketseat.com.br), nas aulas do evento.

## Como funciona

1. **Cole seu código** no editor
2. **Ative o modo brutal** (se tiver coragem)
3. **Clique em roast** e espere o resultado
4. Receba sua nota, comentários sarcásticos e sugestões reais de melhoria
5. Se for ruim o bastante, você entra no **leaderboard de vergonha** 🏆

## Funcionalidades

- **Code Roasting** — análise de código com IA que mistura humor ácido e feedback técnico real
- **Score 0-10** — nota visual em anel animado com cores por faixa (verde, amarelo, vermelho)
- **Modo Brutal** — toggle para aumentar a intensidade do roast
- **Shame Leaderboard** — ranking público dos piores códigos já submetidos
- **Syntax Highlighting** — destaque de sintaxe com tema Vesper
- **Diff View** — mostra o que deveria mudar no seu código

## Rodando localmente

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).
